## 📊 현재 프로젝트 상태 (v2.6.0)

### ✅ 완료된 주요 기능

#### 투표 시스템
- 2주 단위 투표 기간 자동 생성
- 글 추천 및 투표
- 중복 투표 가능
- 우승자 자동 선정

#### 철학챗 시스템 ⭐ 최신 업데이트
- **정기 모임**: 매주 정해진 요일과 시간에 진행
- **즉흥 모임**: 취미/토론 모임으로 세분화
- **필터링**: 즉흥 모임, 정기 모임, 지난 모임(관리자 전용)
- 카카오톡 오픈채팅 연동
- 실시간 채팅
- 참석 체크 시스템
- 모임 참여 횟수 추적

#### 오늘의 질문 시스템
- 일일 질문 자동 생성
- 개인 답변 (비공개)
- 공개 답변 작성
- 이미지 업로드 (최대 2장)
- 댓글 시스템

#### 인증 시스템
- 초대 코드 기반 회원가입
- 3단계 역할 (관리자/모임장/일반 회원)
- 비로그인 사용자도 홈/철학챗 조회 가능

#### 알림 시스템
- 웹 기반 알림 벨
- 채팅 메시지 알림
- 알림 클릭 시 자동 삭제

#### 관리자 기능
- 회원 관리
- 투표 관리
- 모임 관리
- 초대 코드 관리
- 커뮤니티 통계

## 🗂️ 파일 위치

### 주요 소스 코드
```
src/
├── components/          # 재사용 가능한 컴포넌트
│   ├── common/         # 공통 UI
│   ├── meetings/       # 모임 컴포넌트
│   ├── voting/         # 투표 컴포넌트
│   └── admin/          # 관리자 컴포넌트
├── pages/              # 페이지 컴포넌트
│   ├── meetings/       # 모임 페이지
│   │   ├── CreateMeetingPage.jsx    ⭐ 모임 생성
│   │   ├── MeetingsPage.jsx         ⭐ 모임 목록
│   │   └── MeetingDetailPage.jsx    ⭐ 모임 상세
│   ├── voting/         # 투표 페이지
│   ├── questions/      # 질문 페이지
│   └── admin/          # 관리자 페이지
├── utils/              # 유틸리티 함수
├── lib/                # 외부 라이브러리 설정
├── store/              # 전역 상태 관리
└── hooks/              # 커스텀 훅
```

## 💡 팁

### 문서 검색 방법
1. **파일명으로 검색**: `MEETING`, `FEATURES`, `STATUS` 등
2. **내용으로 검색**: 특정 기능명이나 에러 메시지로 검색
3. **날짜로 검색**: 마이그레이션 파일은 YYYYMMDD 형식

### 문서 업데이트 규칙
1. 새로운 기능 추가 → PROJECT_SUMMARY.md + FEATURES.md 업데이트
2. 데이터베이스 변경 → MIGRATION_GUIDE.md 업데이트
3. 버그 수정 → CURRENT_STATUS.md 업데이트
4. 큰 기능 추가 → 별도 업데이트 문서 생성 (예: MEETING_TYPES_UPDATE.md)

### 버전 관리
- **Major 버전 (X.0.0)**: 큰 기능 추가 또는 구조 변경
- **Minor 버전 (0.X.0)**: 새로운 기능 추가
- **Patch 버전 (0.0.X)**: 버그 수정 또는 작은 개선
