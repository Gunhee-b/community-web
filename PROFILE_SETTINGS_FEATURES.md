# 프로필 및 설정 화면 기능 요약

> 현재 웹 및 앱 애플리케이션의 프로필/설정 기능 정리 (2025-12-03 기준)

---

## 📱 프로필 페이지

### Web (`/web/src/pages/ProfilePage.jsx`)

#### 기본 정보 카드
| 항목 | 설명 |
|------|------|
| 사이트 닉네임 | 표시 + 수정 버튼 |
| 역할 | 관리자 / 모임장 / 일반 회원 (배지 형태) |
| 가입일 | yyyy년 MM월 dd일 형식 |

#### 90-Day Challenge 카드
| 항목 | 설명 |
|------|------|
| 총 체크 수 | 질문 체크 총 횟수 |
| 연속 체크 일수 | 현재 연속 참여 일수 |
| 90일 달성률 | 퍼센트 + 프로그레스 바 |
| 최장 연속 기록 | 역대 최장 연속 일수 |
| 남은 일수 메시지 | "목표까지 N일 남았어요!" 또는 "90일 챌린지 완료!" |

#### 활동 통계 카드
| 항목 | 설명 |
|------|------|
| 내가 쓴 글 | 공개 답변 개수 |
| 모임 참여 | 참여한 모임 횟수 |
| 투표 참여 | 미구현 (-) |
| 글 추천 | 미구현 (-) |

#### 닉네임 수정 모달
- 새 닉네임 입력 (2-20자)
- 한글, 영문, 숫자 사용 가능
- 중복 닉네임 불가
- 취소 / 변경 버튼

### App (`/app/app/(tabs)/profile.tsx`)

#### 프로필 헤더 카드
| 항목 | 설명 |
|------|------|
| 아바타 | 그라데이션 원형 (닉네임 첫 글자) |
| 닉네임 | 표시 + 수정 버튼 |
| 역할 | 관리자(빨강) / 모임장(보라) / 일반회원(파랑) 배지 |
| 가입일 | yyyy년 M월 형식 |

#### 90-Day Challenge 카드
| 항목 | 설명 |
|------|------|
| 총 체크 수 | 파란색 박스 |
| 연속 체크 일수 | 초록색 박스 |
| 90일 달성률 | 퍼센트 표시 |
| 최장 연속 기록 | 일수 표시 |
| 프로그레스 바 | 그라데이션 (파랑→초록) |
| 남은 일수 메시지 | 동적 메시지 |

#### 활동 통계 카드
| 항목 | 색상 | 설명 |
|------|------|------|
| 내가 쓴 글 | 주황 | 공개 답변 개수 |
| 모임 참여 | 보라 | 참여 횟수 |
| 투표 참여 | 파랑 | 미구현 (-) |
| 글 추천 | 초록 | 미구현 (-) |

#### 관리자 전용 섹션
- 관리자 대시보드 바로가기 (role === 'admin' 일 때만 표시)
- 빨간색 방패 아이콘

#### 하단 버튼
- 로그아웃 버튼 (빨간색)

#### 기타
- 상단 우측: 설정(톱니바퀴) 아이콘 → `/settings` 이동
- 닉네임 수정 모달 (웹과 동일 기능)
- 다크 모드 지원

## ⚙️ 설정 페이지

### App (`/app/app/settings.tsx`)

#### 화면 설정
| 항목 | 설명 |
|------|------|
| 테마 | 라이트 / 다크 선택 버튼 |
| 언어 | 현재 한국어 (변경 불가) |

#### 알림 설정
| 항목 | 타입 | 설명 |
|------|------|------|
| 푸시 알림 | Switch | 전체 푸시 알림 ON/OFF |
| 모임 알림 | Switch | 모임 관련 알림 |
| 질문 알림 | Switch | 질문 관련 알림 |
| 채팅 알림 | Switch | 채팅 관련 알림 |
| 알림 소리 | Switch | 알림 소리 ON/OFF |

#### 정보
| 항목 | 동작 |
|------|------|
| 이용약관 | `/terms` 페이지로 이동 |
| 개인정보처리방침 | `/privacy` 페이지로 이동 |
| 버전 정보 | Alert: "Rezom Community v1.0.0" |

#### 계정 관리
| 항목 | 설명 |
|------|------|
| 차단 목록 | `/blocked-users` 페이지로 이동 |
| 계정 삭제 | 삭제 확인 모달 표시 |

#### 계정 삭제 모달
- 경고 아이콘 (노란색)
- 삭제되는 데이터 목록:
  - 모든 개인정보
  - 작성한 답변 및 글
  - 90-Day Challenge 기록
  - 모임 참여 기록
- "계정삭제" 입력 확인 필요
- 취소 / 삭제 버튼

#### 로그아웃
- 확인 Alert 후 로그아웃 실행

## 🎨 UI/UX 특징

### App 디자인 특징
- **색상 시스템**: 역할별 색상 구분 (관리자-빨강, 모임장-보라, 일반-파랑)
- **통계 박스**: 각 항목별 배경색 차별화 (파랑, 초록, 주황, 보라)
- **프로그레스 바**: 그라데이션 효과 (파랑→초록)
- **다크 모드**: 전체 UI 다크 테마 지원
- **카드 기반 레이아웃**: 섹션별 카드로 구분

### Web 디자인 특징
- **심플한 레이아웃**: 카드 기반 정보 표시
- **라이트 모드만**: 다크 모드 미지원
- **통계 그리드**: 2x2 그리드 형태
