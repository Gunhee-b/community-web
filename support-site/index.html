자주 묻는 질문 (FAQ)

Q1. 로그인이 안 돼요

A. 지원되는 로그인 방법:

  • Apple로 로그인
  • Google로 로그인
  • Kakao로 로그인

네트워크 연결을 확인하시고, 문제가 계속되면 위 이메일로 문의해 주세요.

Q2. 비밀번호를 잊어버렸어요

A. Rezom은 소셜 로그인(Apple, Google, Kakao)만 지원하므로 별도의 비밀번호가 없습니다. 각 서비스의 계정으로 로그인해 주세요.

Q3. 계정을 삭제하고 싶어요

A. 앱 내에서 직접 계정을 삭제하실 수 있습니다:

  • 앱 실행 → 설정 → 계정 관리 → 계정 삭제
  • 삭제 시 모든 데이터가 즉시 삭제됩니다
  • 서버 백업 데이터는 30일 이내 자동 파기됩니다

Q4. 부적절한 콘텐츠를 발견했어요

A. 부적절한 콘텐츠를 발견하신 경우:

  • 각 게시물 우측 상단의 ⋮ (더보기) 메뉴를 탭하세요
  • "신고하기"를 선택하고 신고 사유를 선택하세요
  • 신고된 콘텐츠는 24시간 이내에 검토됩니다
  • 부적절하다고 판단된 콘텐츠는 즉시 삭제되며, 작성자는 이용이 제한됩니다

Q5. 다른 사용자를 차단하고 싶어요

A. 원하지 않는 사용자를 차단할 수 있습니다:

  • 차단하고 싶은 사용자의 프로필로 이동하세요
  • 프로필 화면에서 "차단하기" 버튼을 탭하세요
  • 차단된 사용자의 콘텐츠는 더 이상 보이지 않습니다
  • 차단 목록은 설정 화면에서 관리할 수 있습니다

Q6. 앱이 작동하지 않아요

A. 다음 단계를 시도해 보세요:

  • 앱을 완전히 종료한 후 다시 실행
  • 기기 재시작
  • App Store에서 최신 버전으로 업데이트
  • 인터넷 연결 확인

문제가 계속되면 ingk.tech@gmail.com으로 문의해 주세요.
