# Rezom 앱 기능 정리서

## 개요

Rezom은 철학적 토론과 커뮤니티 활동을 위한 모바일 앱입니다. 일상 질문, 오프라인 모임, 투표, 실시간 채팅 등의 기능을 제공합니다.

---

## 1. 인증 및 사용자 관리

### 로그인
| 기능 | 설명 |
|------|------|
| Google 로그인 | OAuth 2.0 기반 |
| Kakao 로그인 | OAuth 기반 |
| Naver 로그인 | OAuth 기반 |
| Apple 로그인 | iOS 네이티브 인증 |
| 자동 로그인 | AsyncStorage 토큰 유지 |

### 계정 관리
- 프로필 편집 (유저네임 변경)
- 계정 삭제
- 로그아웃

---

## 2. 홈 화면 (대시보드)

### 구성 요소
| 섹션 | 내용 |
|------|------|
| 오늘의 질문 배너 | 매일 새로운 질문 표시, 답변 개수 |
| 예정된 모임 | 가장 가까운 모임 카드 표시 |
| 투표 배너 | 월간 베스트 투표 기간 표시 |
| 최근 활동 | 새 모임/질문 피드 |
| 토스트 알림 | 실시간 채팅 알림 |

---

## 3. 철학챗 (모임)

### 모임 목록
- **캐주얼 모임** (⚡): 가벼운 만남
- **정규 모임** (📅): 정기적인 토론

### 모임 카드 정보
- 모임 이미지/그라디언트 배경
- 장소 및 일시
- 제목 및 설명
- 참가자 현황 (현재/최대)
- 호스트 정보 (아바타, 이름, 뱃지)

### 참가 기능
| 기능 | 설명 |
|------|------|
| 참가하기 | 모임 참가 신청 |
| 나가기 | 참가 취소 |
| 실시간 채팅 | 모임 내 대화 |
| 카카오톡 오픈채팅 | 외부 링크 연결 |

### 모임 상세
- 전체 모임 정보
- 참가자 목록
- 실시간 채팅
- 신고/차단 기능

## 4. 질문 & 답변

### 질문
| 항목 | 설명 |
|------|------|
| 오늘의 질문 | 매일 새로운 철학적 질문 |
| 지난 질문 | 과거 질문 아카이브 |
| 답변 개수 | 각 질문별 답변 수 표시 |

### 답변 기능
- 텍스트 답변 작성
- 이미지 첨부 (최대 2장)
- 공개/비공개 설정
- 답변 수정

### 댓글
- 답변에 댓글 작성
- 댓글 삭제
- 실시간 댓글 피드

## 9. 네비게이션 구조

### 하단 탭
```
홈 → 철학챗 → 질문 → 프로필
```

### 화면 구조
```
├── (auth)
│   ├── login
│   └── signup
├── (tabs)
│   ├── index (홈)
│   ├── meetings (모임 목록)
│   ├── questions (질문 목록)
│   └── profile (프로필)
├── meetings/[id] (모임 상세)
├── questions/[id] (질문 상세)
├── answers/[id] (답변 상세)
├── settings (설정)
├── notifications (알림)
├── blocked-users (차단 목록)
├── privacy (개인정보처리방침)
└── terms (이용약관)
```

## 10. 기술 스택

| 분야 | 기술 |
|------|------|
| 프레임워크 | React Native + Expo Router |
| 상태관리 | Zustand (persist) |
| 백엔드 | Supabase (PostgreSQL) |
| 실시간 | Supabase Realtime |
| 인증 | OAuth (Google, Kakao, Naver, Apple) |
| 푸시알림 | Expo Notifications |
| 이미지 | Expo Image Picker |
| 저장소 | AsyncStorage |
| UI | Expo Vector Icons, LinearGradient |

## 12. 핵심 사용자 플로우

### 1) 신규 가입
```
앱 실행 → 소셜 로그인 선택 → OAuth 인증 → 홈 화면
```

### 2) 질문 답변
```
홈 → 오늘의 질문 → 답변 작성 → 이미지 첨부 → 공개/비공개 선택 → 제출
```

### 3) 모임 참가
```
철학챗 탭 → 모임 선택 → 상세 보기 → 참가하기 → 실시간 채팅
```

### 4) 사용자 신고
```
콘텐츠 메뉴 → 신고하기 → 사유 선택 → 상세 입력 → 제출
```
